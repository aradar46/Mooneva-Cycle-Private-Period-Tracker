# Mooneva Cycle — Period & Cycle Tracker

A private, open-source menstrual cycle tracker for Android and iOS.

## Features

- Period and cycle tracking
- Calendar view
- Offline-first, no account required
- Privacy-focused — your data never leaves your device

## Building from Source

### Requirements

- Node.js 18+
- npm
- Capacitor CLI

### Install dependencies

```bash
npm install
```

### Run in browser

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Android

```bash
npx cap sync android
npx cap open android
```

### iOS

```bash
npx cap sync ios
npx cap open ios
```
